# Impeller Speed

- **Flow**:
=+=
$$ { {q_1 \over q_2} = {n_1 \over n_2} } $$
=+=

=+=
$$ q = flow\ capacity\ (m^3,\ /s,\ gpm\ cfm\ etc.) $$
=+=

=+=
$$ n = Revolutions\ per\ minute\ (rpm) $$
=+=

- **Pressure or Head**:
=+=
$$ { {dp_1 \over dp_2} = { ({n_1 \over n_2)})^2 } } $$
=+=

=+=
$$ dp = pressure\ or\ head\ (psi,\ ft,\ m,\ etc.) $$
=+=

- **Power**:
=+=
$$ { {P_1 \over P_2} = { ({n_1 \over n_2)})^3 } } $$
=+=

=+=
$$ P = Power\ (hp,\ w,\ etc.) $$
=+=
